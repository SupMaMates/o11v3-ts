# O11V3-TS

Modified version of **O11V3** with added support for `.ts` stream links, improving IPTV compatibility and playback reliability.

## 🚀 Features

* ✅ Native support for `.ts` stream links
* ⚡ Improved playback stability
* 📡 Better compatibility with IPTV providers
* 🔧 Simple install & uninstall scripts

## 📦 Installation

Run the following command:

```bash
curl -fsSL https://raw.githubusercontent.com/SupMaMates/o11v3-ts/refs/heads/main/install.sh | bash
```

## 🗑️ Uninstallation

To remove O11V3-TS, run:

```bash
curl -fsSL https://raw.githubusercontent.com/SupMaMates/o11v3-ts/refs/heads/main/uninstall.sh | bash
```
